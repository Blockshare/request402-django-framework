Request402

HTTP(S) Request & Response Micropayments Service

Request402.com is a free JSON API for current and historical foreign exchange rates published by the European Central Bank.

The rates are updated daily around 4PM CET.

Usage

Get the latest foreign exchange reference rates in JSON format.

https://www.request402.com/ip
