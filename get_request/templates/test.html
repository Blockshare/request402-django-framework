Request402

HTTP(S) Request & Response Micropayments Service

Request402.com is a JSON API that can be access via micropayments lets you know when something is not right with your digital assets.

Hosted in HTTP + HTTPS

Run curl /info for text instructions on the command line.

created and maintained by cponeill


Basic Requirements:

1. A 21 Bitcoin Computer or 21 installed.
2. You will need to be connected the 21 Zerotier Marketplace.
3. Make sure you have mined or have enough satoshi in your wallet for transactions.

How to use:

Get the origin of your IP address.

Cost: 5 satoshi
$ 21 buy https://www.request402.com/ip
{ "origin": xxx.xxx.x.xx }

Return GET data.

Cost: 5 satoshi
$ 21 buy https://www.request402.com/get?args="example text"
{ "headers": { "args": "example text", "User-Agent": "xxxxxx", "HTTP-Host": "www.xxxxxx.com", "Accept": "*/*", "Encoding": "gzip" }, "origin": "xxx.xxx.xx.xxx" }

Return header and server location data.

Cost: 100 satoshi
$ 21 buy https://www.request402.com/server-location
{ "headers": { "encoding": "gzip", "accept": "*/*", "User-Agent": "xxxxxx" }, "server": { "loc": "xx,xx", "ip": "xx.xx.xx.xxx", "city": "Ashburn", "org": "AS14618 Amazon.com, Inc.", "hostname": "ec2.x.x.x.xxx", "region": "Virginia", "postal": "20149", "country": "US" } }

Get the header, status, and trustworthiness of a specific domain URL.

$ 21 buy https://www.request402.com/domain_status?url=example.com

Use the Request402 Library to display data in the browser.

# Example Flask App from flask import Flask from two1.wallet import Wallet from two1.bitrequests import BitTransferRequests app = Flask(__name__) requests = BitTransferRequests(Wallet()) @app.route('/') def fetch_api(): response = requests.get('https://www.request402.com/server-location') ip_addr = response.json()['server']['ip'] region = response.json()['server']['region'] return ("Your IP address is %s and the server is located in %s." % (ip_addr, region)) if __name__ == '__main__': app.run('::', port=5001)
