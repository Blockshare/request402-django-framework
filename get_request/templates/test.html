Request402

HTTP(S) Request & Response Micropayments Service

Request402.com is a JSON API & Micropayments Service that lets you know when something is not right with your or your clients website.

How to use:

Get the origin of your IP address.

21 buy https://www.request402.com/ip
{ "origin": xxx.xxx.x.xx }

Use Request402 in your own apps and even the browser.

# Example Flask App from two1.wallet import Wallet from two1.bitrequests import BitTransferRequests requests = BitTransferRequests(Wallet()) def fetch_api(): response = requests.get('https://www.request402.com/ip') ip_addr = response.json()['origin'] print("Your IP address is %s." % ip_addr) if __name__ == '__main__': fetch_api()
