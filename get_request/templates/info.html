    You can easily use Request402 by running any of the following commands:


    All endpoints cost between 5 and 2500 satoshi


    /ip: Returns origin IP data: 21 buy https://www.request402.org/ip

    /get: Returns GET data: 21 buy https://www.request402.org/get

    /server-location: Returns server location data: 21 buy https://www.request402.com/server-location?url=example.com

    /get_domain-status: Returns a websites HTTP status: 21 buy https://www.request402.org/get_domain status?uri=example.com

    /get_domain-ip: Returns a websites IP: 21 buy https://www.request402.org/get_domain_ip?uri=example.com

    /ssl-cert: Returns the public key of an HTTPS enabled url: 21 buy https://www.request402.com/ssl-cert?url=example.com

    /company-contact: Returns company contact info: 21 buy https://www.request402.com/company-contact?url=example.com

    /twitter: Returns social profiles and demographic data: 21 buy https://www.request.402.com/twitter?username=exmaple

    /bitcoin: Returns Bitcoin wallet info: 21 buy https://www.request402.com/bitcoin?address=example
